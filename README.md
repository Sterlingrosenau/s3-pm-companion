# S3 PMP Study Companion — 800-Item Model

This GitHub Pages package contains 800 original-to-S3 practice items built from 100 decision archetypes and eight deliberate assessment variants per archetype.

## Blueprint balance

- People: 264 items (33%)
- Process: 328 items (41%)
- Business Environment: 208 items (26%)
- Predictive: 320 items (40%)
- Adaptive/agile and hybrid: 480 items (60%)

## Formats

- 400 single-response scenarios
- 100 multiple-response items
- 100 case-study items
- 100 graphic-based items
- 100 sequence/drag-style items

Automated validation checks the item count, unique IDs and combined prompts, blueprint distribution, format coverage, answer positions, multiple-response keys, sequence keys, explanations, and exam lenses. The items are independent practice material, not PMI questions, and have not been psychometrically calibrated by PMI.

## Publish on GitHub Pages

Upload the contents of this folder to the top level of the repository's `main` branch. Confirm that `index.html` appears directly in the main file list. In **Settings → Pages**, publish from `main` and `/(root)`.

Progress is stored locally in the learner's browser.
